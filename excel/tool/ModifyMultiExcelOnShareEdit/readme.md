# ModifyMultiExcelOnShareEdit

## 背景
* エクセル編集を共有フォルダ上で編集する。
* ファイルが大きくなると重たくて作業が捗らない。
* 複数ファイルに跨った編集は、非常に手間と神経を使う。

## 使用するソフトウェア・使用言語
* powershell
* c#
* html
* css
* javascript

## フロー
1. 準備(リスト作成)
2. 編集箇所の特定
3. 編集開始(リネーム)
4. 編集実行
　4-1. 値置き換え(一部ではなく値全体)
　4-2. かい
5. 編集終了(保存・アップロード・リネーム)
